# Welcome To Books Store Java Web Project

### You will find in this repo the following stuff:
* Java Web source code used in create of Books Store
* JPA (Java Persistence API)
* MVC 2 (Model-View-Controller 2 pattern)
* SQL Server
* JSP/Servlet
